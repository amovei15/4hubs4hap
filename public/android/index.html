Скачайте приложение

Возьмите Happ из магазина приложений или скачайте apk-файл по ссылке из релиза.

При установке apk Android попросит разрешить установку из этого источника — подтвердите.

Запустите приложение и при первом старте разрешите создание подключения.

Вставьте ключ

Добавьте конфигурацию ссылкой или отсканируйте QR-код прямо из приложения.

Что потребуется

  • Android 8.0 и новее
  • Разрешение на установку из выбранного источника (для apk)
  • Конфигурация из личного кабинета

Установка из apk безопасна?

Да, если файл взят из официального источника. Скачивайте apk только по ссылкам из проверенных релизов и не используйте сторонние «моды» — в них может быть лишний код.

Не подключается после установки? Чаще всего причина в неправильно добавленном ключе или выбранном сервере — разбор в разделе Ошибки.
